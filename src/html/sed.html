SED

SEt Decimal mode

Operation: Operation: 1 => D

Mode Syntax HEX LEN TIME
Implied SED $F8 1 2
N V — B D I Z C
        ✓      

Puts the microprocessor in Decimal Mode (for using Binary coded decimal numbers, BCD). In this mode each decimal digit is represented by 4 bits, in a way that one byte stores two decimal digits, namely a number between 00 and 99. When working in this mode a carry is automatically generated between bit 3 and 4 when the result is greater than 9 instead of $0F like it happens normally in the standard binary mode.

CLD, restablishes the normal binary mode of operation.

PusH Processor status on stack

Operation: Push P into the stack



Mode Syntax HEX LEN TIME
Implied PHP $08 1 3
N V — B D I Z C
               

Puts the contents of the Processor Status register (P) into the stack. Later when we need it again we can retrieve this value with the instruction PLP.

P register contains the current state of all the system flags.

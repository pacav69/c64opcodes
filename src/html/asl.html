ASL

Arithmetic Shift Left

Operation: C <= 7 6 5 4 3 2 1 0 <= 0

The negative status flag is set if the result is negative, i.e. has its most significant bit set.
The zero flag is set if the result is zero, or cleared if it is non-zero.
The carry flag is set or cleared depending on the result.

Shifts the bits of the accumulator or a memory address specified by the operand, one position to the left. Bit 7 goes to the Carry flag (C). Zero is shifted into bit zero.

It's the inverse operation of the instruction lsr.

Performing an asl is equivalent to performing a multiplication by 2.

It is worth noting that the 6502 (and 6510) does not have any instruction to perform multiplications.

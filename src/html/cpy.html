CPY

ComPare memory and index Y

Operation: Compare Y and M

Mode Syntax HEX LEN TIME
Immediate CPY #$44 $C0 2 2
Zero Page CPY $44 $C4 2 3
Absolute CPY $4400 $CC 3 4
N V — B D I Z C
✓           ✓ ✓

The negative status flag is set if the result is negative, i.e. has it's most significant bit set.
The zero flag is set if the result is zero, or cleared if it is non-zero.
The carry flag is set or cleared depending on the result.

Compares the contents of a memory address (the next byte of the instruction in Immediate addressing) with the index register Y, updating the flags acordingly with the result (only N, Z and C are affected).

For the comparison, internally the Y index register is substracted from the operand (Y-M). Both register and what is inside the memory address remain untouched (the result is ignored) but the important here is that the appropiate flags are updated reflecting the result of the comparison. Then conditional branching instructions may be used to branch to different points of our program depending of that result.

Example:

    lda #$25       // a = $25
    cpy value      // Modify N, Z and C accordingly to the
                   //  result of the "substraction".
    beq equal      // Branch if equal (if Z = 1)
    bcs greater    // Branch if greater (if C = 1)
    bcc less       // Branch if less (if C = 0)
    ...

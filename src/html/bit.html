BIT

Test bits in memory with accumulator

Operation: A & M, M7 => N, M6 => V



Mode Syntax HEX LEN TIME
Zero Page BIT $44 $24 2 3
Absolute BIT $4400 $2C 3 4
N V — B D I Z C
✓ ✓         ✓  

The negative status flag is set if the result is negative, i.e. has its most significant bit set.
The overflow status flag is set if the operation results in an overflow.
The zero flag is set if the result is zero, or cleared if it is non-zero.

This instruction is a logic version of the compare instructions in which the contents of the accumulator is not modified but the flags are adjusted according to the obtained result.

Bit 6 (Overflow) and bit 7 (Negative) of accumulator is transferred to the P register if the result of A & M = 0 then Z = 1, in other case Z = 0.

Is a form of testing important bits in the accumulator without affecting its contents, as would happen if we did this with and.

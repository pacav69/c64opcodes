JAM - KIL - HLT

These instructions freeze the CPU

Only a reset will restart execution

$F2 JAM CPU lock-up 1

JAM crashes the microprocessor. When one of these opcodes is executed, the byte following the opcode will be fetched, data and address bus will be set to $FF (all 1s) and program execution ceases. No hardware interrupts will execute either. Only a reset will restart execution. This opcode leaves no trace of any operation performed! No registers or flags affected.

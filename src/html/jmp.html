JMP

JuMP to new location

Operation: (PC+1) => PCL, (PC+2) => PCH



Mode Syntax HEX LEN TIME
Absolute JMP $5597 $4C 3 3
Indirect JMP ($5597) $CC 3 5
N V — B D I Z C
               

The Program Counter (PC) is loaded with the address specified by the operand It is the only branch instruction that can use indirect addressing. In this mode the position indicated by the operand in the instruction contains the address to be effectively jumped to. In the operative system of the C64 (called the Kernal) this addressing mode is used so the user can change vectors in RAM to call the system routines. Also this is exploited to patch certain routines into our customized ones easily. You could use this for example to add new basic commands, like Simon's Basic did.

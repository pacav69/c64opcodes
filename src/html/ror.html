ROR

Rotate One bit Right - memory or accumulator

Operation: C => 7 6 5 4 3 2 1 0 => C



Mode Syntax HEX LEN TIME
Accumulator ROR $6A 1 2
Zero Page ROR $44 $66 2 5
Zero Page,X ROR $44,X $76 2 6
Absolute ROR $4400 $6E 3 6
Absolute,X ROR $4400,X $7E 3 7
N V — B D I Z C
✓           ✓ ✓

The negative status flag is set if the result is negative, i.e. has its most significant bit set.
The zero flag is set if the result is zero, or cleared if it is non-zero.
The carry flag is set or cleared depending on the result.

Displaces anticlockwise the bits of the accumulator or a memory address (given by the operand) inversely as what ROL instruction does.

The contents of the Carry flag is put into the bit 7, and bit 0 is moved to the Carry. This results in a circular displacement of 9 bits.

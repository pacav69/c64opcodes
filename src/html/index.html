How to use c64 Opcodes Extension

To execute the extension run the command "Open C64 Opcodes List" from the command pallete. If the cursor is on an opcode when run, that opcodes detailed information will be displayed, otherwise this menu page will load allowing you to browse the opcode lists.

Note: The opcode text does not need to be selected, simply place the cursor on the opcode then run the command.

The source code examples in the descriptions are in Kick Assembler format. Note that you can not copy/paste these examples. VSCode's Webview currently does not allow this unfortunately.

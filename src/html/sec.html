SEC

SEt Carry Flag

Operation: 1 => C



Mode Syntax HEX LEN TIME
Implied SEC $38 1 2
N V — B D I Z C
              ✓

Sets Carry flag (C) = 1.

This instruction is primarily necessary to facilitate the use of the instruction SBC, or to force relative branches using BCC and BCS.

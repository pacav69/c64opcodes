DCM (DCP)

DECs a memory location then CMPs the result with the Accumulator

Sub-instructions: CMP, DEC

Mode Syntax HEX LEN TIME
Zero Page DCM $44 $C7 2 5
Zero Page,X DCM $44,X $D7 2 6
(Indirect,X) DCM ($44,X) $C3 2 8
(Indirect),Y DCM ($44),Y $D3 2 8
Absolute DCM $4400 $CF 3 6
Absolute,X DCM $4400,X $DF 3 7
Absolute,Y DCM $4400,Y $DB 3 7
N V — B D I Z C

Here is an example of how you might use this opcode: 

      dcm $FF         // C7 FF

Here is the same code using equivalent instructions. 

      dec $FF
      cmp $FF

SLO

Shift left one bit in memory, then OR accumulator with memory

Sub-instructions: ORA, ASL

Here is an example of how you might use this opcode: 

      slo $C010         // 0F 10 C0

Here is the same code using equivalent instructions. 

      asl $C010
      ora $C010

LAX (LAX immediate)

Store * AND oper in A and X

(A OR CONST) AND oper -> A -> X

Mode Syntax HEX LEN TIME
Immediate LXA #$AB $AB 2 2
N V — B D I Z C
✓           ✓  

Operation:
This opcode ORs the A register with CONST, ANDs the result with an immediate value, and then stores the result in both A and X.

Here is an example of how you might use this opcode: 

    lxa #$FE         // AB FE

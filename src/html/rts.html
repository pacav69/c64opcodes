RTS

ReTurn from Subroutine

Operation: Saves PC in stack, PC+1 => PC



Mode Syntax HEX LEN TIME
Implied RTS $60 1 6
N V — B D I Z C
               

This instruction works closely with the instruction JSR.

With this command the end of a subroutine is expresed, and causes the return address to be pulled off the stack. Note that since there are instructions to manipulate the stack that return address could be changed. Also one RTS without his matching JSR could be used to jump to the address (plus 1) stored in the stack.
